## ДНІ 1–2 — Бренд та контент

### Крок 5. Логотип та фірмовий стиль

**Інструменти:** Claude (генерація ідей) + Figma / Canva / Adobe Express

1. Відкрити Claude (`claude.ai` або через API):
   ```
   Prompt: «Я розробляю логотип для [назва бізнесу], [тип бізнесу], місто [місто].
   Стиль: [з брифу]. Запропонуй 3 концепції логотипу з описом кольорів,
   шрифту та символіки. Формат: назва концепції, кольори HEX, опис символу.»
   ```
2. Обрати концепцію → намалювати у Canva або Figma
3. Узгодити з клієнтом (надіслати 2–3 варіанти у Telegram)
4. Фіналізувати: PNG (прозорий фон) + SVG

**Фірмовий стиль:** зберегти palette.md:
```
Основний колір: #F97316
Акцент: #F5B942
Фон: #0D0D0F або #FFFFFF
Шрифт заголовків: [назва]
Шрифт тексту: [назва]
```

### Крок 6. Сайт (лендинг)

**Інструмент:** Claude Code (Клодик) — генерація HTML/CSS

**Промпт для Claude:**

```
Створи красивий одностраничний лендинг для бізнесу:

Контакт: телефон [з брифу], Telegram @[нікнейм]

Секції: hero (УТП + CTA), послуги, як це працює, ціни, відгуки, контакти.
Зроби мобільну адаптацію, додай кнопку Telegram і форму заявки.
Зберегти як ~/clients/[назва-клієнта]/index.html
```

**Чек-лист сайту:**
- [ ] Відкривається на мобільному
- [ ] Всі ціни відповідають брифу
- [ ] Телефон клікабельний (`tel:`)
- [ ] Telegram посилання працює
- [ ] Форма заявки відправляє лід в Telegram клієнта
- [ ] Немає placeholder тексту («Lorem ipsum», «Назва компанії»)
- [ ] Швидко завантажується (< 2 сек)

**Деплой:** GitHub Pages (як в репозиторії startai)

```bash
git init && git add . && git commit -m "Initial"
```

## ДНІ 3–4 — AI чат-бот та соцмережі

### Крок 7. Налаштування чат-бота

**Базова версія (шаблон `~/business-bot/`):**

1. Скопіювати шаблон:
   ```bash
   cp -r ~/business-bot ~/clients/[назва-клієнта]-bot
   cd ~/clients/[назва-клієнта]-bot
   ```

2. Відредагувати `config.py` — підставити дані з брифу:
   - `BUSINESS_NAME` — назва бізнесу
   - `BUSINESS_DESCRIPTION` — адреса, телефон, графік
   - `SERVICES` — послуги та ціни з брифу
   - `FAQ` — типові питання (придумати 5–7 з Claude)
   - `OWNER_CHAT_ID` — Telegram ID клієнта (попросити у клієнта: @userinfobot)
   - `BOT_TOKEN` — новий бот від @BotFather для клієнта

3. Придумати FAQ через Claude:
   ```
   Prompt: «Ти — [тип бізнесу] у [місті]. Придумай 7 типових питань
   від клієнтів та відповіді. Послуги: [список]. Формат: Питання / Відповідь.»
   ```

4. Запустити та протестувати:
   ```bash
   source venv/bin/activate
   python main.py
   ```

5. Надіслати клієнту посилання на бота і попросити протестувати

**Для клієнтів з CLAUDE_API_KEY:**
- Додати ключ у `.env`
- Бот автоматично перемикається на AI-режим

### Крок 8. Google Business + Facebook

**Google Business (30 хв):**
1. Зайти на `business.google.com`
2. Знайти / створити профіль бізнесу
3. Заповнити: назва, категорія, адреса, телефон, сайт, графік
4. Додати фото (логотип + фото офісу/команди)
5. Надіслати клієнту код підтвердження (приходить листівкою або дзвінком)

**Facebook сторінка (45 хв):**
1. Створити сторінку з категорією бізнесу
2. Обкладинка + аватар (логотип)
3. Про нас: опис, сайт, телефон, графік
4. 2–3 перших пости (через Claude):
   ```
   Prompt: «Напиши 3 пости для Facebook сторінки [назва].
   Тон: [з брифу]. Мова: українська. Кожен пост до 150 слів + заклик до дії.»
   ```

## ДНІ 5–6 — Реклама та перші ліди

### Крок 9. Запуск таргету

**Facebook Ads — мінімальний старт:**
1. Відкрити Ads Manager → Створити кампанію
2. Ціль: **Ліди** (Lead Generation) або **Трафік** на сайт
3. Аудиторія (з брифу):
   - Місто: [місто клієнта] + 20 км
   - Вік: [з брифу]
   - Інтереси: [відповідні до ніші]
4. Бюджет: від 100–200 грн/день (входить у пакет)
5. Текст оголошення — через Claude:
   ```
   Prompt: «Напиши текст Facebook реклами для [тип бізнесу], [місто].
   Болі аудиторії: [з брифу]. УТП: [з брифу]. Довжина: 3–4 речення + заклик.»
   ```
6. Зображення: логотип + текст (Canva)

**Контроль:** перевіряти ліди кожні 12 годин у перший день.
